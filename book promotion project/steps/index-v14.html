PathWave Signal Generation

Signal Studio for simplifying signal creation

Signal Studio is signal creation software that runs on a PC. It enables the creation of application specific test signals at baseband, RF and microwave frequencies. Study this technically pertinent manual that Electronics Review described as “highly relevant” and the IEEE called “a must read for all hardware engineers .” Pre-order your paperback copy now for just $14.99.

PathWave Signal Generation has two kinds of applications as Desktop or Embedded. PathWave Signal Generation Desktop is running in the PC like Signal Studio, which can create and download generated waveforms into various signal generators. PathWave Signal Generation Embedded is fully integrated application in Keysight VXG signal generator’s firmware. PathWave Signal Generation Embedded shares same measurement technology as PathWave Signal Generation Desktop while its GUI is optimized for touch-based operations.
